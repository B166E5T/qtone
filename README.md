# Qtone Next-Style Compose

This is a clean Jetpack Compose Android TV / FireStick foundation for Qtone.

Direction:
- NexTv-like cinematic style and navigation feel
- Original Qtone layout and branding
- Kotlin + Compose
- ExoPlayer / Media3 only for a smaller APK
- OkHttp Xtream Codes API
- HTTP cleartext enabled for the provided test server
- Left sidebar, rails, focus scaling, matte dark UI

Open the extracted folder directly in Android Studio and build `app`.
